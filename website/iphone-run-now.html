iPhone 15 Pro Max - run now

Phone only * ~10 min * do steps in order

Tap a green link to jump to Settings or the App Store. On iOS 18 some links open the parent pane only - follow the path in each step.

Top 5 - start here

  1. Update iOS - Settings -> General -> Software Update. Open General
  2. Passcode + Face ID + Stolen Device Protection - Settings -> Face ID & Passcode -> Custom Alphanumeric Code -> Stolen Device Protection -> On. Open Face ID & Passcode
  3. Find My iPhone - Settings -> [your name] -> Find My -> Find My iPhone -> On. Open Apple ID
  4. Two-factor authentication - Settings -> [your name] -> Sign-In & Security -> Two-Factor Authentication -> On -> review Devices. Open Apple ID
  5. Safari - Fraudulent Website Warning -> On * Prevent Cross-Site Tracking -> On * Hide IP Address -> From Trackers and Websites. Open Safari settings (iOS 18)

Install apps

  1. Malwarebytes Mobile Security - App Store -> Search -> Malwarebytes Mobile Security -> Get. Open in App Store
  2. Malwarebytes SMS - Settings -> Apps -> Messages -> Unknown & Spam -> Malwarebytes -> On. Open Messages (iOS 18)
  3. Malwarebytes Safari - Settings -> Apps -> Safari -> Extensions -> Malwarebytes -> On. Open Safari (iOS 18)
  4. Cloudflare 1.1.1.1 - App Store -> Search -> 1.1.1.1 Faster Internet -> Get -> allow VPN profile. Open in App Store
  5. Cloudflare DNS - In app: connect On -> Settings -> 1.1.1.1 for Families -> On. One DNS/VPN app only.

Extra minutes

  1. Automatic updates - General -> Software Update -> Automatic Updates -> On.
  2. Mail privacy - Apps -> Mail -> Protect Mail Activity -> On.
  3. Bluetooth & Local Network - Privacy & Security -> revoke unneeded apps. Open Privacy
  4. Lock screen - Face ID & Passcode -> Allow Access When Locked -> trim * Notifications -> sensitive apps -> When Unlocked.
  5. USB Restricted Mode - Face ID & Passcode -> USB Accessories -> Off when locked.
  6. Profiles - General -> VPN & Device Management -> remove unknown profiles.
  7. AirDrop - General -> AirDrop -> Contacts Only.
  8. Messages - Filter Unknown Senders -> On.

Optional

  1. Lockdown Mode - Privacy & Security -> Lockdown Mode -> On. Skip unless high-threat. Open Privacy
