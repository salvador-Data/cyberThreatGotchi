Story

Why we built Cipherhorn

Homelab and small-shop defenders need evidence-grade logging without a six-figure SOC stack. CyberThreatGotchi is an open-source edge IPS on Banana Pi BPI-R3 Mini: live capture, scoring, optional ClamAV, iptables blocks on authorized networks only, SQLite threat history, and a tamper-evident audit chain.

Cipherhorn on e-ink gives humans a mood signal while auditors get structured logs. Kickstarter funds the first honest production run from Philadelphia - flash, burn-in, PETG enclosures, and partner-fulfillment deposits for Meshtastic bundles.

Authorized use

CyberThreatGotchi is a defensive edge sensor for networks you own or are explicitly authorized to monitor. IPS blocks, wardrive profiles, and lab tools require written scope - not opportunistic targeting.

CrackBot, CYD, and Cardputer tools are separate product lines with distinct SKUs and firmware. Reward tiers label each bundle clearly.

FAQ

Quick answers

Is this a real IPS? Yes - capture, scoring, optional ClamAV, iptables block on Linux, SQLite + audit chain.

Can I run without hardware? Clone from GitHub and use --simulation --web or Docker.

What's partner fulfillment? Selected SKUs (Meshtastic) ship from verified independent makers we batch-order after the campaign - not from our Philadelphia bench.

Shipping? Not included in pledge for direct Philly kits; US zone estimates $9-$18. Meshtastic tier includes partner shipping in pledge.
