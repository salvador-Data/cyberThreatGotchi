Built a **layered defensive stack** for a founder workstation and homelab: PowerShell SOC automation (`harden_windows.ps1`, Sysmon + SwiftOnSecurity, Wazuh agent via env-only manager config, Harden-Windows-Security audit-first), resilient **selective SSD + OneDrive** backups with honest fallbacks when D: is offline, and **VPN-preserving** hardening (DuckDuckGo WireGuard exclusions—no stacked DNS VPN installers). Documented **UAC/740/Sysmon** and **No Media** SSD ops; iPhone 15 Pro Max hardening under real iOS limits (Settings + Malwarebytes SMS/Safari without replacing existing VPN/DNS). Pocket **UTMS** on M5 OS Cardputer complements desktop telemetry. CyberThreatGotchi ties it together with DevSecOps policy (`SECURITY_HARDENING.md`), pytest/bandit CI, and payments secrets hygiene—authorized defensive use only.
