## Step 1 — Create or claim the profile

1. Go to [Google Business Profile](https://business.google.com/).
2. Sign in with the Google account you use for Search Console (recommended).
3. **Add business** → enter **Hacker Planet LLC**.
4. Choose category (primary):
   - **Computer security service** (preferred), or
   - **Computer consultant**
5. Add secondary categories if offered:
   - **Computer repair service** (only if you want hardware repair visibility — optional)
   - **Software company** (optional)
