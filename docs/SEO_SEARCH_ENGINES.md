## 2. Bing Webmaster Tools

Bing powers **Bing search**, much of **DuckDuckGo**, and **Yahoo** results.

### Verify hackerplanet.dev

1. Sign in at [Bing Webmaster Tools](https://www.bing.com/webmasters).
2. **Add a site** → `https://hackerplanet.dev`
3. Verification options:
   - **DNS CNAME** (recommended on Cloudflare) — add the CNAME Bing provides
   - **Meta tag** — set `bingSiteVerification` in `website/seo/site.json` to the content value (not the full tag), then run `python scripts/sync_seo.py`. Example placeholder shape:
     ```html
     <meta name="msvalidate.01" content="YOUR_BING_VERIFICATION_CODE"/>
     ```
     Store only `YOUR_BING_VERIFICATION_CODE` in JSON — it is not a secret, but keep it in config for easy rotation.

### Submit sitemap

Same URL as Google:

```
https://hackerplanet.dev/sitemap.xml
```

Bing Webmaster Tools → **Sitemaps** → Submit.

### IndexNow (Bing + participating engines)

We ship an IndexNow key file generated by `sync_seo.py`:

- Key name: `indexNowKey` in `site.json` (default: `hpl-hackerplanet-indexnow-key`)
- Public file: `https://hackerplanet.dev/hpl-hackerplanet-indexnow-key.txt`

After content deploy:

```powershell
python scripts/ping_indexnow.py --dry-run   # preview payload
python scripts/ping_indexnow.py             # POST to api.indexnow.org
```

Optional CI step: run `ping_indexnow.py` after `sync_website_to_docs.py` on main deploys.
